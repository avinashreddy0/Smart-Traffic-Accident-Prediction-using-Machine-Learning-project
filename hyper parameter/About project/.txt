***** Smart Traffic Accident Prediction System *****

This project aims to predict road accident risk using traffic, weather, road, and environmental conditions. The main objective is to identify high-risk situations before accidents occur and support data-driven traffic safety decisions.

The project follows a complete end-to-end Data Science workflow:

Collected and analyzed traffic accident data.
Performed ETL (Extract, Transform, Load) operations.
Cleaned missing values and removed inconsistencies.
Loaded processed data into MySQL Workbench.
Performed SQL analytics to identify accident patterns.
Conducted Exploratory Data Analysis (EDA).
Applied feature engineering to create risk-related features.
Built preprocessing pipelines using Scikit-Learn.
Trained Logistic Regression, Random Forest, and XGBoost models.
Evaluated models using Accuracy, Precision, Recall, F1-Score, and ROC-AUC.
Performed Cross Validation and Hyperparameter Tuning.
Used SHAP for model explainability and feature importance analysis.
Developed an interactive Power BI dashboard with KPIs and insights.
Created a Streamlit application for real-time accident risk prediction.
Enabled users to input traffic conditions and receive predictions.
Visualized accident trends by weather, road conditions, and traffic density.
Identified key factors affecting accident occurrence.
Generated actionable business insights from data.
Integrated SQL, Machine Learning, Power BI, and Streamlit into one solution.
Delivered a complete end-to-end accident risk prediction system.
